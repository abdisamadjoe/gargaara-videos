# Gargaara Testimonials Videos

This repository hosts **self-hosted testimonial videos** for the [Gargaara website](https://gargaara.com/).  

These videos feature partner banks and client feedback, demonstrating the impact of Gargaara financing on businesses, job creation, and economic growth. They are intended for embedding in the homepage **Bento-style testimonial section** using a custom HTML5 player like **Plyr** or **Video.js**.

## Files

- Each video file is clearly named for easy reference:  
  - `testimonial1.mp4`  
  - `testimonial2.mp4`  
  - `testimonial3.mp4`
